Movie API

This page will give you information about last updated movies around the world including: Movie Title, Genre, Release Year, Actors and description.

Request Method URL Response
List of all movies to user GET /movies List of all available movies with image and title
Return available data about specific movie GET /movies/[movie title] A JSON object with movie title, cover art, description, genre, featured film, directors, actors
Return a list of movies by genre GET movies/[genre] Returns and a list of movies that match the specific genre
Return Directors Biography GET /directors/[directors info] Gives information about specific director
Sign up a new user POST /user Allows a user to register a new account
User Login / User can login to their account
View users information GET /users/[username] User can view their account details
Delete Account DELETE /users/[username] Allows a user unregister and delete their account
Update a users info PUT /users/[user-name] Update a users information
Add to favorites PUT /users/[username]/Movies/[Movie ID] Adds a movie to the users favorites list
Remove from Favorites DELETE /users/[username]/Movies/remove/[MovieID] Removes a movie from the users favorites list
